- **Total Duration:** 12 Months (52 Weeks)

# Recommended Programming Languages

Students should be proficient in at least one of:

- Java (Recommended for Hadoop ecosystem)
- Scala (Recommended for Apache Spark)
- Python (Widely used for analytics and data engineering)

---

# Expected Learning Outcomes

By the end of the program, students should be able to:

- Understand distributed systems and distributed storage
- Develop Java-based Hadoop applications
- Build and administer Hadoop clusters
- Process large-scale batch and streaming data
- Design enterprise ETL/ELT pipelines
- Work with Hadoop, Spark, Hive, Kafka, HBase, and NoSQL databases
- Deploy Big Data solutions on AWS, Azure, or Google Cloud
- Optimize cluster performance and resource utilization
- Implement security, governance, and monitoring
- Design and deploy production-grade Big Data architectures
